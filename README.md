# 🎮 Number Guessing Game (1 - 100)

An interactive, logic-based Python game that challenges the user to guess a secret number generated by the computer, providing directional feedback ("Up" / "Down") to guide the player.

## 🚀 Features
* Dynamic random number generation using Python's `random` module.
* Live hint system to narrow down guesses efficiently.
* Attempt counter to track your score.
